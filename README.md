# JAVA-TREE-BASED-GUI
Multi-Functional GUI Application

This repository contains the source code and components for a multi-functional GUI application developed in Java Swing. The application provides a graphical user interface for managing hierarchical data structures, attributes, and groupings. It's designed to be intuitive, modular, and versatile, making it a powerful tool for various data management tasks.

Features
JCheckBoxTree Component: Custom Checkbox Tree Component that enhances JTree with checkbox functionality for node selection and management.

CustomTreeCellRenderer: Custom Tree Cell Renderer to display icons based on node data, providing contextual information.

Groups Management: Panel for managing node groups within the hierarchical structure, improving organization and visualization.

AttributesTableModel: Table Model for managing node attributes in a tabular format with support for adding, deleting, and displaying attributes.

UniqueAttributesManager: Singleton Manager for tracking and displaying unique attribute names for nodes.

ProjectWindow: Main GUI Panel for the Application, providing user-friendly interaction with the hierarchical tree structure and its components.

Internal Class: Entry point for launching internal frames that host various functionalities concurrently, promoting modularity.

How to Use
Clone the repository to your local machine.
Open the project in your preferred Java IDE.
Navigate through the codebase to understand different components and their functionalities.
Compile and run the Internal.java class to launch the application.
